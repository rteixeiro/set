Basic Workstation Setup:

  MS Windows Setup
  Create Local SET Account
  Uninstall Trial Software
  Rename computer (check nable & AD on FSCCDC01 - DHCP)
  Print Label
  Remove bloatware (like the Dell junk that gets installed)
  Power Settings - sleep = never
  Install n-able, AV, and screen connect
  Install Windows Updates

Additional Workstation Setup :

  Install Adobe Reader
  Install Java
  If office365 is not installed already, install it (activate with email license)
  Join computer to domain. After joining to domain, remember to move computer to correct OU in Active Directory.
  Setup user profile (create shortcuts, map drives, set defaults....etc)
  Configure email in Outlook
